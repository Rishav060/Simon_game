# 🎮 Simon Game

A simple memory-based game built using HTML, CSS, and JavaScript.

## 📌 About
This project is a browser-based implementation of the classic Simon memory game.  
The game generates a random color sequence, and the player must repeat it correctly to advance levels.

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation)

## 🎯 Features
- Random sequence generation
- Level progression
- Game over feedback
- Score tracking
